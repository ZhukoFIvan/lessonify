## 1. Тарифы

### Почему 499 ₽/мес?
- Средний чек одного урока: 1 000–2 500 ₽
- 499 ₽ = стоимость ~20 минут работы репетитора
- Порог достаточно низкий чтобы не думать, но достаточно высокий для устойчивого бизнеса
- При 1 000 платящих пользователей = 499 000 ₽/мес

## 2. Что происходит при превышении лимита FREE

### Кейс: у репетитора 10 учеников, он добавляет 11-го

**Вариант А (мягкий, рекомендуемый):**
- Разрешить добавить ученика, но показать модалку: "Вы превысили лимит бесплатного плана. Через 7 дней добавление учеников будет заблокировано. Перейдите на Pro."
- Grace period 7 дней — репетитор не теряет данные и учеников
- После 7 дней — блокируется **только добавление новых** учеников. Все существующие продолжают работать
- Периодические напоминания в Telegram и на дашборде

**Вариант Б (жёсткий):**
- Сразу блокировать добавление 11-го ученика
- Модалка: "Лимит 10 учеников достигнут. Перейдите на Pro."
- Минус: репетитор может уйти к конкуренту

**Рекомендация:** Вариант А. Мягкий подход конвертирует лучше — человек уже использует продукт, привык, ему проще заплатить 499 ₽ чем переносить всё в другое место.

### Кейс: у репетитора PRO истёк, а учеников > 10

- План понижается до FREE
- Все ученики остаются, данные не теряются
- Блокируется добавление новых учеников (пока > 10)
- Блокируется доступ к Pro-фичам (Google Calendar, страница записи, чат)
- Баннер: "Ваш Pro-план истёк. У вас 15 учеников — для продолжения работы со всеми перейдите на Pro."
- Уроки, расписание, оплаты — продолжают работать для ВСЕХ учеников (не удаляем данные и не блокируем существующие уроки)

## 3. Кейсы оплаты

### 3.1 Первая покупка Pro
1. Репетитор нажимает "Перейти на Pro" → открывается платёжная форма
2. Выбирает тариф: месяц (499 ₽) или год (3 990 ₽)
3. Оплачивает картой/СБП
4. `plan=PRO`, `planExpiresAt=+30/365 дней`
5. Если есть реферер → создаётся `ReferralEarning` (20% = ~100 ₽)

### 3.2 Продление подписки
**Автопродление (рекуррентные платежи):**
- При первой оплате сохраняем карту (токен)
- За 3 дня до истечения — уведомление: "Pro продлится автоматически через 3 дня. Отменить?"
- В день истечения — списываем с сохранённой карты
- Если списание не прошло — 3 попытки с интервалом в день
- Если все попытки неудачны — понижаем до FREE

**Ручное продление:**
- За 7 дней — баннер "Скоро истекает"
- За 3 дня — push + Telegram
- В день истечения — понижаем до FREE
- Можно оплатить заново в любой момент

**Рекомендация:** начать с ручного продления (проще в реализации), добавить авто позже.

### 3.3 Промокод + оплата
- Промокод даёт бесплатные дни
- Если Pro был оплачен и промокод применён → дни суммируются
- Промокод не возвращает деньги

### 3.4 Возврат
- В течение 3 дней после оплаты — полный возврат по запросу в поддержку
- После 3 дней — возврат не производится (стандартная политика SaaS)
- Возврат делается через API платёжной системы

## 4. Платёжные системы

### Требования:
- Работа для самозанятого (НПД) без ИП
- Приём карт (Visa/MC/Мир) и СБП
- Поддержка рекуррентных платежей (подписки)
- Webhook для уведомлений о платежах
- Node.js SDK или REST API

### Сравнение (актуальные данные на март 2026):

| **Robokassa** | ✅ Да | от 3.5% | 0.4% | ✅ Да | Бесплатно | Бесплатные авто-чеки «Робочеки СМЗ» через «Мой налог». npm-пакеты. |

### Рекомендация:

**1-й выбор: Prodamus**
- Заточен под самозанятых в образовании/коучинге
- Подписки и рекуррентные платежи из коробки
- REST API v3, JS-библиотека, webhook-уведомления
- 3-месячный промо-период: 0% комиссия на СБП (до 100К оборота)
- Активация за 24 часа
- Подключение: 2 500 ₽ (разовая плата)

**2-й выбор: Robokassa**
- Бесплатное подключение, от 3.5% карты
- Бесплатные авто-чеки «Робочеки СМЗ» → автоматически в «Мой налог»
- npm-пакеты для Node.js
- Проверенная платформа, давно на рынке

**3-й выбор: Paymaster**
- Самая низкая комиссия: от 2.95% карты, 0–0.7% СБП
- Поддерживает СМЗ с 2019 года
- Нет офиц. SDK, но простой REST API
- Требует больше документов при подключении (паспорт, ИНН, СНИЛС, справка НПД)

**Будущее (после регистрации ИП):** перейти на Tinkoff Payments — самая низкая комиссия (2.49%).

## 5. Техническая реализация (Prodamus)

### Флоу оплаты:

```
1. Репетитор нажимает "Перейти на Pro"
2. Frontend → POST /payments/create-subscription
3. Backend создаёт ссылку на оплату через Prodamus API
4. Redirect на страницу оплаты Prodamus
5. Пользователь платит
6. Prodamus отправляет webhook → POST /payments/webhook
7. Backend: обновляет plan=PRO, planExpiresAt, создаёт ReferralEarning
8. Redirect обратно в приложение → "Подписка оформлена!"
```

### Новые таблицы:

```prisma
model Subscription {
  id              String   @id @default(cuid())
  userId          String
  provider        String   // prodamus
  externalId      String   @unique // ID платежа в Prodamus
  plan            String   // monthly, yearly
  amount          Int      // 499 или 3990
  status          String   // active, cancelled, expired, failed
  currentPeriodEnd DateTime
  cancelledAt     DateTime?

  createdAt DateTime @default(now())
  updatedAt DateTime @updatedAt

  user User @relation(fields: [userId], references: [id], onDelete: Cascade)

  @@index([userId])
  @@map("subscriptions")
}

model PaymentEvent {
  id             String   @id @default(cuid())
  subscriptionId String?
  provider       String
  externalId     String   @unique
  type           String   // payment, refund, failed
  amount         Int
  rawData        Json     // полный ответ от платёжки

  createdAt DateTime @default(now())

  @@index([subscriptionId])
  @@map("payment_events")
}
```

### Endpoints:

```
POST   /subscription/create    → создать ссылку на оплату
POST   /subscription/webhook   → webhook от Prodamus (без авторизации, проверка подписи)
GET    /subscription/status    → текущий статус подписки пользователя
POST   /subscription/cancel    → отменить автопродление
```

## 6. Налоги (самозанятый)

- Ставка НПД: 4% (от физлиц) или 6% (от юрлиц/ИП)
- В нашем случае: 4% от каждого платежа
- Prodamus может автоматически формировать чеки и отправлять в "Мой налог"
- При обороте > 2.4 млн ₽/год — нужно регистрировать ИП

### Экономика при 499 ₽/мес:
- Получаем: 499 ₽
- Комиссия Prodamus (3.8%): -19 ₽
- Налог НПД (4%): -20 ₽
- Чистыми: ~460 ₽ с одного пользователя
- *Первые 3 мес при оплате через СБП: комиссия 0%, чистыми ~479 ₽*
